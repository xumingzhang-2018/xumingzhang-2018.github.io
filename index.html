#GitHub+Hexo 搭建个人网站

##一、搭建博客

###1、下载Git for Windows并安装。
下载地址：https://github.com/waylau/git-for-win

###2、下载Node.js并安装。

下载地址：https://nodejs.org/en/download

###3、安装Hexo博客框架工具
在自己新建的目录（最好新建一个，以后自己Blog的所有东西都存放在这里）下单击右键，选择“Git Bash Here”，打开Git Bash， 执行如下npm命令即可安装。

$ npm install -g hexo-cli
$ hexo init <folder>
$ cd <folder>
$ npm install
